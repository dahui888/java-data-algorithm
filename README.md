## 常见错误：
### 1. 查找最大回文字符串：
* 相邻字符是回文的判断逻辑不对
* 长度为2个以上串的内循环注意事项（有两个指针）

### 2. 链表删除倒数第n个节点
* 删除倒数第n个，要找到第n+1个,否则会把倒数n-1个删除掉

### 3. 合并两个有序链表
* 循环完毕，注意了不要把后走完的节点丢失了

### 4. 二叉树完全相等
* 最后的return 注意额，值判断和递归在一行上
